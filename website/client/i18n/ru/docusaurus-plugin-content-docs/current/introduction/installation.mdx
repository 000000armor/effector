## Совместимость

:::note
Добавлено в effector 20.1.0
:::

Используй `effector/compat`, `effector-react/compat` и `effector-vue/compat` модули для поддержки IE11 и Chrome 47 (браузер Smart TV устройств) без необходимости компиляции через babel.

```diff
- import {createStore} from 'effector'
+ import {createStore} from 'effector/compat'
```
